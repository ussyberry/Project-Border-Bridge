# BorderBridge Document Templates - Formatting Guide

This guide provides consistent formatting standards to be applied across all BorderBridge business document templates to ensure brand alignment and professional presentation.

## Brand Visual Elements

### Colors
- **Primary Blue (#1A5F7A)**: Use for headings, section dividers, and important information
- **Secondary Teal (#57C5B6)**: Use for subheadings, highlights, and secondary elements
- **Accent Orange (#F8A145)**: Use sparingly for call-to-action elements and important highlights
- **Neutral Charcoal (#2D3E40)**: Use for body text
- **Light Background (#F9F7F7)**: Use for document background or alternating table rows

### Typography
- **Headings**: Montserrat Bold, 16-20pt for main headings, 14-16pt for subheadings
- **Body Text**: Open Sans Regular, 11-12pt
- **Emphasis**: Open Sans Semibold for emphasis within body text
- **Spacing**: 1.15-1.25 line spacing for optimal readability

### Logo Placement
- Position the BorderBridge logo in the top-right corner of the first page
- Use the full-color logo on white background
- Maintain minimum clear space around the logo (equal to the height of the "B" in BorderBridge)

## Document Structure

### Headers and Footers
- **Header**: Include document title, client name, and document ID/reference number
- **Footer**: Include page numbers, date of issue, and BorderBridge contact information
- Use a thin horizontal line (1pt, Primary Blue) to separate headers/footers from content

### Margins and Layout
- **Margins**: 1 inch (2.54 cm) on all sides
- **Alignment**: Left-aligned text for body content
- **Section Spacing**: 18-24pt space before new sections
- **Paragraph Spacing**: 12pt after paragraphs

### Tables and Lists
- **Tables**: Use thin borders (0.5pt, Secondary Teal), with slightly darker header row
- **Lists**: Use custom bullets that incorporate the bridge motif or standard bullets in Primary Blue
- **Numbered Lists**: Use Primary Blue for numbers

## Tone and Language Guidelines

### Voice Characteristics
- **Knowledgeable but Approachable**: Demonstrate expertise without being intimidating
- **Empathetic and Supportive**: Acknowledge emotional aspects while providing practical guidance
- **Confident but Humble**: Project certainty without overpromising
- **Global Perspective**: Demonstrate awareness of different cultural contexts
- **Forward-thinking and Optimistic**: Focus on opportunities while being realistic

### Language Standards
- Use "you" and "your" to speak directly to clients
- Avoid jargon; when technical terms are necessary, explain them
- Use active voice rather than passive voice
- Maintain a slightly formal tone that remains warm and accessible
- Frame immigration as an opportunity rather than a challenge

### Document-Specific Tone Adjustments

#### Proposal
- Emphasize possibility and opportunity
- Be specific about value and benefits
- Use confident, forward-looking language

#### Service Agreement
- Use clear, precise language
- Maintain professionalism while avoiding overly complex legal jargon
- Balance necessary legal protection with accessibility

#### Onboarding Guide
- Use warm, welcoming language
- Provide clear, actionable instructions
- Emphasize support and guidance

#### Invoice/Receipt
- Use straightforward, professional language
- Be transparent about all charges
- Include a warm thank you message

#### Feedback Form
- Use inviting, appreciative language
- Frame questions positively
- Emphasize the value of client input

## Visual Elements and Graphics

### Icons and Symbols
- Use consistent icon style across all documents
- Incorporate bridge imagery where appropriate
- Use icons to improve navigation and highlight key information

### Images and Graphics
- Use high-quality, professional images that reflect diversity
- Include data visualizations where appropriate (charts, graphs)
- Maintain consistent style for all visual elements

### Dividers and Borders
- Use subtle dividers between sections (thin lines in Primary Blue or Secondary Teal)
- Consider using bridge motif as a decorative element in page breaks or section dividers
- Use borders sparingly to highlight important information

## Digital Document Considerations

### Hyperlinks
- Format all hyperlinks in Secondary Teal with underline
- Ensure all links are descriptive (avoid "click here")
- Include full URLs in parentheses for printed documents

### Form Fields
- Clearly indicate editable fields with subtle highlighting or borders
- Use consistent formatting for all form fields
- Include clear instructions for completion

### Accessibility
- Ensure sufficient color contrast for readability
- Use heading structures properly for screen readers
- Include alt text for all images and graphics

## Implementation Notes

- All templates should be available in both editable formats (DOCX, Google Docs) and PDF
- Maintain consistent naming conventions for all document files
- Include version numbers and last updated dates in document properties
- Create templates that work well both digitally and when printed
