## Formatting and Usage Guidelines

### Email Templates
- Personalize the subject line based on recipient's industry or background
- Always include recipient's name in greeting
- Maintain bullet points and formatting for readability
- Include company logo in email signature
- Ensure all links are trackable for analytics
- Test emails on multiple devices before mass sending

### LinkedIn Templates
- Keep connection requests under LinkedIn's character limit (300 characters)
- Use follow-up messages only after connection is accepted
- Personalize first line with specific reference to recipient's profile
- Avoid excessive use of emojis or special characters
- Include a clear call-to-action in follow-up messages

### Website Landing Pages
- Ensure mobile responsiveness for all landing page elements
- Implement A/B testing for headlines and CTAs
- Include high-quality images relevant to each segment
- Optimize page load speed for better conversion
- Implement tracking pixels for retargeting campaigns
- Ensure form fields capture essential information without friction

### Social Media DMs
- Personalize opening line with reference to specific content
- Keep messages concise and focused on value proposition
- Include clear next step or question to encourage response
- Follow platform-specific best practices for message formatting
- Respect messaging limitations on different platforms

### Follow-up Communications
- Send follow-ups within 48 hours of initial contact
- Include reference to previous interaction
- Provide clear, specific next steps
- Attach relevant materials mentioned in previous communications
- Include multiple contact options for convenience
