# AI Agents Documentation

This repository is designed to be friendly for AI agents.

## Project Structure
- `src/`: Source code for the React application.
- `docs/`: Project documentation, business plans, and strategies.
- `workflows/`: Automation workflows (e.g., n8n).
- `scripts/`: Utility scripts.

## Conventions
- **Code Style**: Functional React components with Hooks.
- **Styling**: Vanilla CSS (imported in components).
- **State Management**: React Context or local state.

## Key Files
- `src/App.jsx`: Main application component and routing.
- `src/main.tsx`: Entry point.
- `docs/executive_summary.md`: High-level project overview.
